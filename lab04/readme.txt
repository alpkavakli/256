1. Install project dependencies:

   npm install

2. Start the application:

   npm run dev

3. Test the following endpoints in your browser:

   http://localhost:3000
   http://localhost:3000/users
   http://localhost:3000/users?page=5&sort=name
   http://localhost:3000/users/1234
   http://localhost:3000/users/421/posts
   http://localhost:3000/users/421/posts?page=3
